# Sensor Data Simulator

## Overview

A Python-based simulation of a temperature monitoring system similar to an embedded sensor application.

## Features

- Generates random temperature values
- Stores sensor readings
- Calculates average temperature
- Provides warning system

## Concepts Used

- Random data generation
- Lists
- Loops
- Basic data analysis

## Future Improvements

- Connect with Arduino
- Add real sensors
- Create graphical dashboard
